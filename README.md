## easy-inputs v1.0.0
---
__Easy__ terminal inputs for an __Easy__ time.

### Installation
---
```
npm install easy-inputs

yarn add easy-inputs
```

### How to use
---

```
const ez = require('easy-inputs');

//String prompt
var name = ez.text('What is your name?: );

//Number prompt
var name = ez.number('What is your age?: );
```
